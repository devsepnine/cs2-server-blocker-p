## ? 자주 묻는 질문

---
- 어떻게 동작하며, 게임에서 밴이 될 수 있습니까?
  - 이것은 CS2의 게임 파일이나 시스템 파일을 조작하지 않으며 단순히 방화벽 정책을 수정하여 서버 접속을 차단하는 프로그램입니다.
- 관리자 권한이 왜 필요합니까?
  - 윈도우의 권한이 필요한 작업을 진행할때 필요한 방화벽 정책입니다. 일반모드에서 서버 블록을 진행할 수 없으므로 관리자 권한 실행을 요청합니다.
